    Qu'est ce que « laboîte maker » ?

    Bienvenue sur la documentation de laboîte maker !

    Vous pouvez consulter directement le guide de montage, mais si vous souhaitez en savoir plus sur le kit, c'est en dessous que ça se passe !

    Info

    Il existe également une version .pdf du guide.

    Composants

    La boîte maker se présente sous la forme d'un kit à monter soi-même composé de :

    • un microcontrôleur Adafruit HUZZAH32 (ou équivalent) ;
    • deux matrices de LED ;
    • un boîtier imprimé en 3D ;
    • un protège écran en PMMA.

    Microcontrôleur Adafruit HUZZAH32 (ou équivalent)

    Ce microcontrôleur est le cerveau de laboîte, il récupère depuis lenuage.io les informations à afficher puis transmet ces données aux matrices de LED.

    Les modèles de microcontrôleur testés sont les suivants :

    Microcontrôleur Prix Testé
    Adafruit HUZZAH32 – ESP32 Feather Board ~24€ ✅
    SparkFun ESP32 Thing ~24€ ✅
    NodeMCU ESP32 ~12€ ⛔️
    LOLIN D32 ~6€ ✅

    Matrices de LED

    Les matrices de LEDs permettent l'affichage des informations envoyées par le microcontrôleur.

    Chacune des 8 matrices est constituée de 64 LED pilotées par un circuit intégré appelé MAX7219. L'écran formé par ces 8 matrices comptera donc 512 LED pilotables individuellement et la luminosité de l'écran pourra être ajusté (16 niveaux de luminosité).

    Les modèles de matrices de LED testés sont les suivants :

    Matrice de LED Prix Testé
    MAX7219 Dot Matrix Module Microcontroller 4 In One Display with 5P Line 4 in 1 Red 2×4€ ✅
    MAX7219 Dot Matrix Module Microcontroller 4 In One Display with 5P Line 4 in 1 Green 2×4€ ✅

    Boîtier imprimé en 3D

    L'écran de laboîte sera encadré par un boitier imprimé en 3D permettant de donner une rigidité à l'ensemble.

    Deux ouvertures sont présentes sur le côté pour laisser passer les fils des matrices et deux encoches sont présentes sur la face avant pour fixer le protège écran en PMMA.

    Info

    Les fichiers pour la fabrication sont publiées sur le site YouMagine (voir les ressources ci-dessous).

    Protège écran en PMMA

    L'écran de laboîte sera protégé par une pièce de PMMA découpée au laser et viendra s'enficher sur le boitier imprimé en 3D.

    Info

    Les fichiers pour la fabrication se trouvent également sur YouMagine (voir les ressources ci-dessous).
